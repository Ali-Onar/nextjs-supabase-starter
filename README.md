# Nextjs Supabase Starter

This project covers the content of the Full Stack Series on my [YouTube channel]((https://www.youtube.com/@alionar60)).

## Overview

In this tutorial we will build a CRUD application by combining Nextjs and Supabase.
We started this project with Supabase template: `npx create-next-app -e with-supabase`
You will also learn in this guide:
 - Client and Server request with Supabase APIs
 - Supabase Auth, APIs, Storage

## Getting Started

- Node.js
- npm or yarn
- Change env.example to .env.local file, and use your api keys from Supabase.

Clone the repository:

```bash
git clone https://github.com/Ali-Onar/nextjs-supabase-starter.git
 ```

SQL Schema Configuration
- Copy sql codes from schema.sql file and paste your Supabase project on SQL Editor

Install dependencies:

```bash
npm install
# or
yarn install
```
Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
